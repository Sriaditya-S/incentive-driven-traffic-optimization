# Incentive-Driven Framework for Sustainable Urban Traffic Optimization

This project presents a novel approach to urban traffic management by combining **eco-routing**, **traffic simulation**, and **blockchain-based incentives** to encourage environmentally responsible driving behavior.

---

## 🚀 Overview

Traditional navigation systems optimize only for travel time, often leading to congestion and increased emissions.  
This project introduces a **"Drive-to-Earn" model**, where users are rewarded for choosing eco-friendly routes.

The system integrates:
- Route optimization (fastest vs eco-friendly)
- Traffic simulation using SUMO
- CO₂ emission estimation
- Trip verification
- Blockchain-based reward distribution

---

## 🧠 Key Idea

Instead of forcing restrictions, the system **incentivizes better decisions**.

> Lower emissions → Verified trip → Earn carbon credit tokens

---

## ⚙️ System Workflow

1. User submits trip request  
2. System generates:
   - Fastest route
   - Eco-friendly route  
3. Traffic simulated using SUMO  
4. Emissions calculated for both routes  
5. Trip verified using GPS / toll data  
6. CO₂ savings computed  
7. Tokens issued via smart contract
